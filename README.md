
GreenStoreBD is a full-featured eCommerce platform for wholesale and retail shopping in Bangladesh. 
It allows users to browse and purchase products imported directly from China, with seamless user authentication using Firebase, a responsive frontend built with Next.js, and a robust backend powered by Express.js and MongoDB. 

Features:
- User authentication with Firebase
- Product management and categories
- Shopping cart and checkout
- SSLCommerz payment integration (sandbox)
- SEO optimized with dynamic metadata and structured data
- Responsive design and fast performance

Tech Stack:
- Frontend: Next.js 16 (Turbopack), React Query, Tailwind CSS
- Backend: Node.js, Express.js, MongoDB
- Authentication: Firebase
- Deployment: VPS with Nginx and PM2
- Cloudinary for media management

This repository contains the **frontend** code for GreenStoreBD.
